Где работает Python?

Python работает не только на Земле, но и на Марсе: он используется в марсоходе, созданном NASA.

С чем работает Python?

С Python работают крупнейшие мировые компании: Google, IBM, Netflix, Spotify, Amazon, Pinterest, YouTube и многие другие.

Почему язык называется Python'ом?

Название языка не имеет ничего общего со змеями, он назван так в честь популярной британской комедийной группы Монти Пайтона из 1970ых.
Python повлиял на JavaScript

Python является одним из 9 языков, которые повлияли на дизайн JavaScript. Другие языки: AWK, C, HyperTalk, Java, Lua, Perl, Scheme и Self.
Python старше, чем думалось

Python старше Java. Первая версия Python появилась в 1991 году, а язык Java увидел свет в 1996 году.
